# Web application that lists all survivors from a zombie apocalypse

### React for zombie

Back-End:<br />
https://github.com/gabriel-cardoso-oliveira/zombie-server<br />

## Quick start

Install:

```
git clone https://github.com/gabriel-cardoso-oliveira/zombie-client.git
```
```
cd zombie-client/
```
```
yarn install
```

Test:

```
yarn run start
```
